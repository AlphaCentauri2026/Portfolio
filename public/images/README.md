# Images Folder

This folder contains all images used in the portfolio website.

## Folder Structure

```
public/images/
├── profile/          # Profile and personal photos
├── projects/         # Project screenshots and images
├── icons/           # Custom icons (if any)
└── backgrounds/     # Background images (if any)
```

## Image Guidelines

- **Profile Photos**: Use JPG or PNG format, recommended size: 300x300px or larger
- **Project Images**: Use JPG or PNG format, recommended size: 800x600px or larger
- **File Names**: Use descriptive names with hyphens (e.g., `profile-photo.jpg`, `travlr-project.png`)

## Recommended Images to Add

1. **Profile Photo**: `profile/corey-hamilton.jpg`
2. **Project Screenshots**:
   - `projects/travlr-screenshot.png`
   - `projects/disney-clone-screenshot.png`
   - `projects/noxx-browser-screenshot.png`

## Usage in HTML

```html
<!-- Profile photo -->
<img src="public/images/profile/corey-hamilton.jpg" alt="Corey Hamilton">

<!-- Project screenshot -->
<img src="public/images/projects/travlr-screenshot.png" alt="Travlr Project">
```
